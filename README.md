## 一、介绍

基于springboot+ vue前后端分离实现的大学生社团活动平台

### 1、学生功能
个人中心 、 社团申请管理 、 活动报名管理、 我的收藏管理等功能

### 2、社长功能

个人中心 、社团信息管理 、 社团申请管理、 社团活动管理 、活动报名管理 等功能

### 3、管理员功能

个人中心、学生管理 、社长管理 、 社团类型管理 、社团信息管理、 社团申请管理、 社团活动管理等功能
